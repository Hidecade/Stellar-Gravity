# Stellar Gravity

**Stellar Gravity** は、重力と合体をテーマにした  
HTML5 + JavaScript（Matter.js）製のブラウザゲームです。

PC・スマートフォンのブラウザでそのままプレイできます。

---

## 🎮 ゲーム概要

- 中心にあるコア（恒星）へ向けて惑星を射出
- 同じ大きさの惑星同士が衝突すると **合体・進化**
- 惑星が危険ラインを超え続けると **GAME OVER**
- 最終進化「Black Hole」に到達すると **STAGE CLEAR**

シンプルな操作と、物理挙動・演出を重視したゲームデザインです。

---

## 🕹 操作方法

### 共通
- **クリック / タップ**：惑星を射出

### BOOST
- **BOOST ボタン**
- **スペースキー（PC）**

回転速度が上がり、射出に横方向のドリフトが加わります。

---

## 🔊 サウンドについて

- BGM / 効果音あり
- **タイトル画面でのみ BGM ON / OFF 切替可能**
- 設定は `localStorage` に保存されます

⚠ iPhone / Safari では  
**最初にユーザー操作（STARTボタン）を行う必要があります**
