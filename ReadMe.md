# Stellar Gravity

**Stellar Gravity** は、重力と天体の進化をテーマにした、Matter.js 物理エンジン搭載のスタイリッシュ・グラビティパズルです。

宇宙の静寂を感じさせるミニマルなデザインと、心地よい物理挙動をブラウザでそのまま体験できます。

---

## 🎮 ゲーム概要

* **コアへの射出**: 自動回転するランチャーから、中心の恒星に向けて惑星をショットします。
* **合体と進化**: 同じ天体同士をぶつけると、より巨大な天体へ進化（全9段階）。
* **コンボシステム**: 連続して合体させるとボーナススコアが発生。白く輝く星屑が舞うエフェクトが爽快感を高めます。
* **重力崩壊**: 最終進化「Black Hole」に到達すればステージクリア。コアが超新星爆発を起こし、次の星系へ移行します。
* **デッドライン**: 惑星が境界線を3秒間超え続けると重力均衡が崩れ、**GAME OVER** となります。

---

## 🕹 操作方法

### 惑星の射出
* **画面上部をクリック / タップ**: 待機中の天体をセンターへ射出します。

### BOOST（加速・偏向）
* **BOOST ボタンをホールド**
* **スペースキー（PC）**
回転速度が劇的にアップし、射出時に強力な横方向のドリフト（遠心力）を加えることができます。

### ユーティリティ
* **RESET**: ゲーム中、いつでも即座にタイトルへ戻れます。iPhone Safari 等のモバイル端末でも一撃で反応するレスポンス設計になっています。

## ✨ 視覚・演出効果

* **ミニマリズム**: 白を基調とした細身のタイポグラフィと、清潔感のある洗練されたUI。
* **グローエフェクト**: 天体やパーティクルが幻想的に光り輝く、高品位な光彩演出を導入。
* **スマートフィードバック**: コンボ発生時には加算スコアが優雅に浮遊し、星屑（パーティクル）となって霧散します。

---

## 🔊 サウンドについて
* 没入感を高めるアンビエントなBGMと、重厚な合体効果音。
* タイトル画面で **BGM ON / OFF** の切替が可能（設定はブラウザに保存）。
* **⚠ iPhone / Safari ユーザーの方へ**: 最初の「START」タップ時にオーディオが自動的にアンロックされます。
